**CLASS**

# `Label`

A widget showing a piece of text.

## Properties
### `label`

The widget for macOS.

### `content`

The content in macOS.

### `label`

The widget for Linux.

## Methods
### `init()`

Initialize a label.

### `setText(_:)`

Set the label's text.
- Parameter text: The new text.
- Returns: The label.

### `setFont(_:)`

Set the label's font.
- Parameter font: The label's font.
- Returns: The label.
